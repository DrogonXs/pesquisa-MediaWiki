MediaWiki

O que é o MediaWiki?

O MediaWiki é um programa extremamente poderoso e escalável, e uma implementação de wikis ricas em funcionalidades interessantes, que utiliza o PHP para processar e exibir os dados armazenados na sua base de dados, tal como MySQL. As páginas utilizam o formato de texto wiki do MediaWiki, para que os usuários sem conhecimento de HTML ou CSS possam editá-las facilmente. Quando um usuário publica uma edição de página, o MediaWiki insere a edição na base de dados sem eliminar as versões anteriores da página, permitindo assim uma fácil reversão da edição em casos de vandalismo ou spamming. Permite também a gestão de imagens e arquivos multimídia, que são armazenados no sistema de arquivos. Para wikis grandes com muitos usuários, o MediaWiki tem cache e pode ser facilmente integrado com servidores proxy. Com extensões dedicadas, o MediaWiki também pode manipular dados estruturados.

Criação do MediaWiki

Fase I: O MediaWiki foi originalmente escrito para o site Wikipédia pelo estudante e programador alemão Magnus Manske, criado em 2001 que inicialmente utilizava o software UseModWiki feito com a linguagem de programação Perl, sendo chamada por "Fase I".

Fase II: Sendo alterado para PHP com suporte ao MySQL na versão "Fase II", em 25 de janeiro de 2002 (data ficou conhecido entre a comunidade Wikipedista como o Dia Magnus) sendo então chamado por "PHP-wiki", "software Wikipedia".

Fase III: Em 2002 o software foi reescrito e melhorado por Lee Daniel Crocker a partir do código base, sendo conhecido por "Fase III". A partir de 2003, a Fase III ou "PHP wiki" adotou o nome de "MediaWiki", batizado por Daniel em homenagem a Fundação Wikimedia. As novas versões públicas foram criadas pensadas na expansão, utilização externa além dos projetos da Wikimedia. Inicialmente o MediaWiki foi usado no site Wikipédia, e os outros projetos da Wikimedia. Como o software é disponibilizado para uso gratúito, atualmente diversos sites são feitos usando a MediaWiki. Muitas das ferramentas usadas na Wikipédia ou que usam o MediaWiki podem ser usadas em outras plataformas semelhantes, que compartilham uma estrutura semelhante à encontrada na MediaWiki

Magnus Manske

Heinrich Magnus Manske (nascido em 24 de maio de 1974) é um cientista sênior no Wellcome Trust Sanger Institute, em Cambridge, no Reino Unido e um desenvolvedor de software de uma das primeiras versões do MediaWiki.

Lee Daniel Crocker

Lee Daniel Crocker (nascido em 3 de julho de 1963) é um programador de computador americano . Ele é mais conhecido por reescrever o software no qual a Wikipédia é executada, para resolver problemas de escalabilidade . Este software, originalmente conhecido como "Fase III", foi lançado em julho de 2002 e se tornou a base do que hoje é chamado de MediaWiki . O repositório de código do MediaWiki ainda era chamado de "phase3" até a mudança do Subversion para o Git em março de 2012.
